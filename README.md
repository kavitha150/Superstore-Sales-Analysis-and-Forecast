# Superstore Sales Dashboard

This project is an Excel-based dashboard created using the Superstore dataset.  
It provides a clear view of sales performance, trends, and forecasting.

## Features
- Total, Average, Max, and Min Sales
- Sales by Product Category
- Top-Selling Sub-Categories
- Sales by State and Customer Segment
- Monthly Sales Trend
- 7-Day Sales Forecast
- Interactive slicers for:
  - Region
  - Category
  - Sub-Category
  - Segment
  - State
  - Ship Mode

## Tools Used
- Excel 
- Pivot Tables
- Pivot Charts
- Forecasting Tools
- Basic Data Cleaning
